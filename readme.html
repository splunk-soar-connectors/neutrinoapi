Use this app for identifying malicious hosts, anonymous proxies, Tor Nodes, botnets, spammers and more.
Block, filter and flag traffic to help reduce attacks on your networks and software.
Blocklists are generated from 3 primary source categories
1) System of automated bots, crawlers and honeypots which continuously collect data from across the Internet.
2) Firewall Aggregation from thousands of commercial and open-source security appliances.
3) Open Data collected from many public sources of IP related data including public blocklists, blacklists, malware/botnet trackers and various security intelligence feeds
